«🚀 DecodeLabs Artificial Intelligence Internship – Project 2»

📖 Overview

This project demonstrates a Supervised Machine Learning Classification Model using the famous Iris Dataset.

The objective is to train a model that can classify iris flowers into different species based on their features using the Decision Tree Classification Algorithm.

---

✨ Features

- 📂 Load the Iris Dataset
- 📊 Split the dataset into Training & Testing sets
- 🌳 Train a Decision Tree Classifier
- 🔍 Predict flower classes
- 📈 Calculate Model Accuracy
- 📋 Display Classification Report
- 🎯 Generate Confusion Matrix
- 💾 Save the Trained Model

---

🛠️ Technologies Used

- 🤖 Scikit-learn
- 💾 Joblib

🧠 Machine Learning Algorithm

🌳 Decision Tree Classifier

A Decision Tree is a supervised machine learning algorithm that learns decision rules from the training data and predicts the correct class for new data. It is simple, fast, and easy to understand, making it an excellent choice for beginners.
