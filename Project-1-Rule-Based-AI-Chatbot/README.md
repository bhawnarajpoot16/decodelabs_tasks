# 🤖 AI Chatbot

A simple and interactive Rule-Based AI Chatbot developed using Python. This project simulates basic human conversation by responding to predefined user inputs using conditional (`if-else`) logic.

---

## 📌 Project Overview

The chatbot is designed to:
- Welcome users with greetings.
- Respond to common predefined questions.
- Continue the conversation until the user exits.
- Demonstrate the fundamentals of Artificial Intelligence using rule-based decision making.

This project was developed as **Project 1** during my Artificial Intelligence Internship at DecodeLabs.

---

## ✨ Features

- 👋 Greets the user
- 💬 Responds to predefined questions
- 🔄 Runs in a continuous conversation loop
- 🚪 Exits gracefully when the user types "bye" or "exit"
- 🧠 Uses simple `if-else` logic for decision making

---

## 🛠️ Technologies Used

- Python 3
- Conditional Statements (if-else)
- Loops
- Basic Input/Output

---

## ▶️ How to Run

1. Install Python 3 on your system.
2. Open the project folder.
3. Run the following command:

```bash
python chatbot.py
```

*(Replace `chatbot.py` with your actual Python file name if it is different.)*
