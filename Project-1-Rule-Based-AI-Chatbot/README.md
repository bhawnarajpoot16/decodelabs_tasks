## 📖 Overview

This project is a simple **Rule-Based AI Chatbot** developed using Python 🐍.

The chatbot interacts with users through text-based conversations and provides responses based on predefined rules using conditional statements.

This project demonstrates the basic concept of Artificial Intelligence by implementing a simple conversation system using logical decision-making.

---

# ✨ Features

- 💬 Interactive text-based conversation
- 👋 Greeting responses
- 🧠 Rule-based response generation
- ❓ Answers predefined user queries
- 🔄 Handles multiple conversation inputs
- 🚪 Provides exit responses

---

# 🛠️ Technologies Used

- 🐍 Python 3
- 🧩 Conditional Statements (if-elif-else)
- 💻 Python Development Environment

The chatbot works using predefined rules and conditional logic.

The workflow:

```

     ↓

     ↓
Bot Reply
```
