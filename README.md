# 🌸 Project 2: Data Classification Using AI 📊

A supervised machine learning classification model developed using the famous **Iris Dataset 🌺**.

The project uses the **Decision Tree Classifier 🌳** algorithm to classify flowers into different species based on their features.

### ✨ Features:
- 📂 Load Iris Dataset
- 📊 Split Training & Testing Data
- 🌳 Train Machine Learning Model
- 🔍 Predict Flower Classes
- 📈 Calculate Model Accuracy
- 📋 Generate Classification Report
- 🎯 Create Confusion Matrix
- 💾 Save Trained Model

### 🛠️ Technologies:
- 🐍 Python 3
- 🤖 Scikit-learn
- 💾 Joblib

📁 Project Folder:
```
AI-Classification
```
